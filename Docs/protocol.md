# Communication Protocol

## USART Configuration

* Baud Rate: 9600
* Data Bits: 8
* Stop Bits: 1
* Parity: None

## Packet Structure

| Byte | Description       |
| ---- | ----------------- |
| 0    | Start Byte (0xFF) |
| 1-12 | Command Data      |
| 13   | Checksum          |

## Checksum

The checksum is generated by summing all command bytes:

```c
checksum = data[0] + data[1] + ... + data[11];
```

## Relay Commands

| Command | Action      |
| ------- | ----------- |
| '1'     | Relay 1 ON  |
| '2'     | Relay 1 OFF |
| '3'     | Relay 2 ON  |
| '4'     | Relay 2 OFF |

## Communication Flow

1. Transmitter reads sensors
2. Packet is generated
3. USART transmits packet
4. Receiver validates checksum
5. Receiver updates outputs
